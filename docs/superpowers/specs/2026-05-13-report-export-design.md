# Report Export Design

> **版本**：v1.0
> **日期**：2026-05-13

---

## 1. 目标

PC 版扫描结果导出为 PDF / JSON / HTML 报告，数据来源为当前 pipeline 结果或历史 SQLite。

---

## 2. 架构

```
PipelineResults / SQLite Session
         │
         ▼
┌──────────────────────────────────────┐
│         ExportPanel (Frontend)         │
│  [PDF 快速] [PDF 正式] [JSON] [HTML] │
└──────────────────────────────────────┘
```

| 方式 | 实现 | 速度 | 可控性 |
|------|------|------|--------|
| PDF 快速 | `window.print()` + print CSS | 快 | 低 |
| PDF 正式 | pdfmake | 慢 | 高 |
| JSON | Blob download | 快 | — |
| HTML | Blob download | 快 | — |

## 5. Tauri Command

```rust
#[tauri::command]
fn export_session_json(session_id: i64) -> Result<String, String> {
    let db = state.db.lock().map_err(|e| e.to_string())?;
    let detail = db.get_session_detail(session_id)?;
    serde_json::to_string_pretty(&detail).map_err(|e| e.to_string())
}
```

前端用 `document.write()` 或 `Blob` download 触发下载。
